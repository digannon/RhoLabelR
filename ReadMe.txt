To run program, compile and execute RhoLabelR.java
On a linux machine, go to the directory containing:
    RhoLabelR.java
    Graph.java
    GraphManager.java
    Node.java
and enter into the command line:

javac RhoLabelR.java
java RhoLabelR


This program is designed to search for a rho-labeling of
an input graph G. 
For more information on rho-labelings:
http://math.illinoisstate.edu/reu/Art063.RhoLabe2-Regs-AJC.pdf
